# 🌐 Advanced IP-Tracer Pro (Kali Linux Edition)

![Python](https://img.shields.io/badge/Python-3.x-blue.svg)
![OS](https://img.shields.io/badge/OS-Linux%20%7C%20Windows-red.svg)

**IP-Tracer Pro** is a professional-grade tool designed to fetch highly accurate geographical data, ISP details, and security insights (Proxy/VPN detection) for any IP address. 

---

## 🔥 Key Features
- ✅ **High Accuracy:** Precise location data including City, Country, and Zip Code.
- ✅ **ISP Intelligence:** Detailed info about Service Providers and Organizations.
- ✅ **Security Audit:** Real-time detection for VPNs and Proxies.
- ✅ **Interactive Mapping:** Instant redirection to Google Maps for visual location.
- ✅ **CLI Optimized:** Sleek, color-coded terminal interface for Kali Linux.

---
